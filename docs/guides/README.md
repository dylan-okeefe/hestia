# Guides

These guides walk through setup, configuration, and day-to-day operation of Hestia.

## Suggested reading order

1. **[Runtime setup](runtime-setup.md)** — Install deps, start llama.cpp, and run your first turn.
2. **[Environment variables](environment-variables.md)** — Reference for every `HESTIA_*` env var.
3. **[Voice setup](voice-setup.md)** — Enable Telegram voice messages (STT + TTS).
4. **[Email setup](email-setup.md)** — Connect IMAP/SMTP so Hestia can read and draft mail.
5. **[Multi-user setup](multi-user-setup.md)** — Run Hestia for more than one person safely.
6. **[Security](security.md)** — Threat model, trust presets, and hardening checklist.
7. **[Reflection tuning](reflection-tuning.md)** — Configure the background analysis loop.
8. **[Telegram conversation audit](telegram-conversation-audit.md)** — Review and export chat history.
