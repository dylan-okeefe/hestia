# v0.9.0 release prep

**Target version:** v0.9.0
**Tag at:** develop tip after the Track 3 hotfix bundle
(`feature/hotfix-v0.9.0-copilot-critical`) lands.
**Written:** 2026-04-20 — retroactive. All six feature branches below were
merged to develop between 2026-04-19 and 2026-04-20 under the prior
(narrower) prep doc `v0.8.x-multi-user-safety-release-prep.md`. Dylan has
since reviewed the full as-merged scope and accepted it as v0.9.0. This
document exists so future audits see authorization by exact branch name
before the tag is placed. The superseded doc is archived at
`docs/development-process/reviews/archive/v0.8.x-multi-user-safety-release-prep.md`.

## In-scope feature branches

- `feature/l40-copilot-cleanup` — Copilot cleanup backlog from v0.8.0 review
  (concurrent tool dispatch, `should_evict_slot` stub removal, email adapter
  hardening, Gmail folder configurability, TODO-marker sweep).
- `feature/voice-shared-infra` — L41 voice pipeline (`VoicePipeline`,
  faster-whisper STT, Piper TTS), `SileroVAD` stub, `VoiceConfig`,
  `hestia[voice]` extra, `hestia doctor` voice check, voice setup guide.
- `feature/voice-phase-a-messages` — L42 Telegram voice-message handler
  gated by `TelegramConfig.voice_messages` (OGG/Opus transcode via
  `ffmpeg`, 1 MB voice-note truncation fallback, text follow-up).
- `feature/l45a-trust-identity-plumbing` — runtime identity `ContextVar`s
  (`current_platform`, `current_platform_user`), per-user `trust_overrides`
  on `HestiaConfig`, `DefaultPolicyEngine._trust_for(session)`, scheduler
  identity inheritance.
- `feature/l45b-memory-user-scope-migration` — one-way FTS5 recreate-and-copy
  migration adding `platform`/`platform_user` to the `memory` table,
  user-scoped `MemoryStore` API (fail-closed), per-session epoch compilation,
  `LIKE`-based fallback when FTS5 unavailable.
- `feature/l45c-multi-user-docs-and-hardening` — `src/hestia/platforms/allowlist.py`
  with shared `match_allowlist()` + Telegram/Matrix validators, adapter
  allow-list refactor, `docs/guides/multi-user-setup.md`, README multi-user
  section.
- `feature/hotfix-v0.9.0-copilot-critical` — pre-tag hotfix bundle (13
  items from the 2026-04-20 Copilot audit — all 6 critical, 5 of 10 high,
  A-3 architecture, T-1 test gap). Scope detailed in
  `v0.9.0-release-and-audit-response.md` Track 3. **Must land and pass
  gates before v0.9.0 is tagged.**
